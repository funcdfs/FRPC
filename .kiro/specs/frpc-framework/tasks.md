# 实现计划: FRPC 框架

## 概述

本实现计划将 FRPC 框架的设计转化为可执行的编码任务。FRPC 是一个基于 C++20 协程的高性能 RPC 框架，包含网络引擎、协程调度器、RPC 客户端/服务端、服务治理等核心组件。

实现将按照依赖顺序进行：首先搭建基础设施和底层组件（网络引擎、协程调度器），然后实现核心 RPC 功能，最后添加服务治理和测试。
